# Veramo

Veramo is a JavaScript Framework for Verifiable Data that enables creation and management of decentralized identifiers and verifiable credentials without worrying about interop and vendor lock-in. The framework is built around a plugin system and runs on Node, Browsers, and React Native straight out of the box. Veramo supports flexible key management, storage, and multiple DID methods for building decentralized identity applications.
