Incognito is a privacy-focused blockchain that provides privacy infrastructure and applications for web3. It operates as a privacy sidechain that can be attached to any blockchain to conduct confidential asset transfers. Uses ring signatures, stealth addresses, and confidential transactions to completely mask sender, receiver, and transaction amounts.
