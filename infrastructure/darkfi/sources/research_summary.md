## Executive Summary

DarkFi is an **active, privacy-focused Layer 1 blockchain** founded by crypto-anarchist Amir Taaki. The project emphasizes anonymous engineering using zero-knowledge cryptography (Halo2), zkVM, and a custom smart contract language (zkas).

**Key Milestone**: Alpha testnet launched with anonymous IRC application (February 2025).

### Products & Applications

1. **darkfid** - Full Node

   - Maintains P2P network connectivity

2. **drk** - CLI Wallet
   - Interface to smart contracts (Money, DAO)
   - Key and coin management
   - Blockchain scanning for balance updates

3. **minerd** - Mining Daemon
   - Connects to darkfid over RPC
