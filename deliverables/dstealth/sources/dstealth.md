# Stealth (XST)

## Basic Project Description

Stealth (XST) is a privacy-focused cryptocurrency that aims to provide fast, feeless, private, and scalable transactions. It operates on its own blockchain called Junaeth and utilizes a Quantum Proof-of-Stake (qPoS) consensus mechanism.
