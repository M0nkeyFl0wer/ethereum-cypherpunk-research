## Description

Aztec is the first decentralized, privacy-preserving L2 on Ethereum. It brings programmable privacy to smart contracts through PLONK-based zk-SNARKs, enabling confidential transactions and private state while maintaining composability with public execution.

### Key Differentiators

- **Client-side proofs**: Data never leaves user devices
- **Noir language**: Rust-like DSL for private smart contracts
- **Day-1 decentralization**: No centralized sequencers or backdoors
- **UTXO private state**: Encrypted notes with nullifiers
