## Overview

STELS Protocol is a privacy-focused DeFi solution that enables users to interact with decentralized finance protocols (like Uniswap) while maintaining complete transaction privacy through stealth addresses and ephemeral relays.

## How STELS Works

According to the project website:

1. User initiates a DeFi transaction (e.g., Uniswap trade)
2. STELS generates a unique stealth address for this transaction
3. An ephemeral relay is created and funded
4. The relay executes the transaction from the stealth address
5. User's main wallet is never exposed on-chain
6. Transaction history is unlinkable, defeating chain analysis
