## ðŸŽ¯ Key Insights

### What We Learned
1. **Data is scattered**: Same information appears in multiple files with different structures
2. **Nested deeply**: Often hidden as `tier_2_data.founders` or `funding_info.investors`
3. **Different naming**: `description` vs `project_overview` vs `overview.description`
4. **Rich metadata**: Most fields have confidence scores and source citations
5. **Quality varies**: `research_result.json` > `*_gemini.json` > `*_scraped.json` > `news_summary.json`

### Best Extraction Strategy
1. **Recursive search**: Use `rglob('*.json')` to find all files
2. **Multiple paths**: Try 3-5 different field names for same data
3. **Type checking**: Fields can be string, dict with `.value`, or arrays
4. **Confidence filtering**: Only use data with confidence > 0.70
5. **Source verification**: Check `.sources` array for reliability
