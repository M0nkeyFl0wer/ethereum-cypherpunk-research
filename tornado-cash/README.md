## ğŸ“ Description

Tornado Cash is an open source, non-custodial, fully decentralized cryptocurrency mixer (tumbler) that runs on Ethereum Virtual Machine-compatible networks. It is a fully decentralized protocol for private transactions on Ethereum, BSC (BNB Chain), Optimism, Polygon, Avalanche, Arbitrum, and Gnosis networks.
