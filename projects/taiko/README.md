## Project Description

Taiko is a Type 1 (Ethereum-equivalent) ZK-Rollup with Based Contestable Rollup (BCR) architecture. It is perfectly compatible with Ethereum, supporting all EVM opcodes without modifications to the Ethereum framework.

**Key Innovation**: First based rollup on Ethereum mainnet with decentralized, permissionless sequencing driven by the layer-1 (Ethereum) blockchain itself.

### Technology Highlights

- **Privacy Mechanism**: Zero-Knowledge Proofs (ZK-SNARKs/STARKs)
- **Architecture**: Type 1 ZK-Rollup (Ethereum-equivalent)
- **Sequencing**: Based Sequencing (fully decentralized)
- **Multi-Proof System**: Supports SNARKs, STARKs, and SGX (Trusted Execution Enclave)

### Core Innovations

1. **Type 1 ZK-EVM**: Perfectly Ethereum-equivalent with no alterations to hash functions, state trees, or gas expenses. Inherits all future Ethereum upgrades automatically.

2. **Based Sequencing**: Sequencer is driven by the layer-1 (Ethereum) blockchain, making it fully decentralized and permissionless.

3. **Based Contestable Rollup (BCR)**: Combines based sequencing with a contestation mechanism and multi-proofs (SNARKs, STARKs, and Trusted Execution Enclave).

4. **Multi-Proof System**: Supports multiple proof types: SNARKs, STARKs, and SGX via the raiko repository.
