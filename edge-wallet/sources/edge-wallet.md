# Edge Wallet

## Basic Project Description

Edge Wallet is a mobile-first cryptocurrency wallet designed for managing, sending, receiving, buying, selling, and exchanging various digital assets. It was initially launched as Airbitz in 2014 and rebranded to Edge in 2017. The wallet is known for its user-friendly interface and strong security features, including client-side encryption.
