- **Algorithm**: Elliptic Curve Integrated Encryption Scheme (ECIES)
- **Purpose**: Encrypt AES-256 keys using receiver public keys and sender private key

**Symmetric Encryption**:
- **Algorithm**: AES-256
- **Purpose**: Content encryption

**Key Management**:
- **Distribution**: Distributed GunDB nodes share AES-encrypted keys
- **Decryption Flow**: User's private key decodes encrypted AES key from GunDB ‚Üí AES key decrypts content
