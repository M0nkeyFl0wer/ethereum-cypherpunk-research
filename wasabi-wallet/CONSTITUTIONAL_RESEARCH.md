# Wasabi Wallet

Wasabi Wallet is an open-source, non-custodial, privacy-focused Bitcoin desktop wallet that implements trustless CoinJoin mixing using the WabiSabi protocol to hide transaction histories through collaborative transactions. All traffic is routed through the Tor network for enhanced anonymity. Wasabi processes over 15,000 mixed transactions monthly and has passed independent security audits with no critical vulnerabilities found in its core privacy protocols.
