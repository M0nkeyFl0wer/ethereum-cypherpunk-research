# Project Metadata - zecrey

**Name**: Zecrey

**Category**: Uncategorized

**Description**: Zecrey is a zk-rollup based Layer 2 privacy protocol that brings cross-chain support and privacy to digital assets. Built on zero-knowledge proof technology, Zecrey enables private transactions and cross-chain asset management across multiple blockchain networks. The protocol uses zk-rollup scaling to provide privacy-preserving transactions while maintaining compatibility with various blockchain ecosystems.
